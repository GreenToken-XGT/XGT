# Green Token
全球首个ESG绿色数字基金
The first ESG green digital fund, an ETF driving ReFi and global impact.

# 1️⃣ 关于 Green Token

Green Token 由全球 RWA 资产上链孵化联盟联合马来西亚 · ESG 公会共同发起，是全球首个 ESG 绿色数字基金。

Green Token 以 ReFi (Regenerative Finance，再生金融) 机制为核心，汇聚全球资本力量，构建可持续、可扩展的全球绿色经济生态体系，推动绿色产业发展，助力实现联合国 17 项可持续发展目标 (SDGs)。

在此基础上，Green Token 的底层架构将基于全球主流的“绿色金融 ETF”运行逻辑，聚焦优质 ESG 企业与绿色资产，为全球资本提供高效、合规、可规模化参与绿色项目的一站式入口。

通过持续领投与生态化运作，Green Token 将分散的绿色资产整合为可配置、可扩展的绿色资产组合，使生态价值、社会价值与经济价值在同一体系内形成稳定的正向循环。
