# Green Token
全球首个ESG绿色数字基金

The first ESG green digital fund, an ETF driving ReFi and global impact.

# 1️⃣ 关于 Green Token

Green Token 由全球 RWA 资产上链孵化联盟联合马来西亚 · ESG 公会共同发起，是全球首个 ESG 绿色数字基金。

Green Token 以 ReFi (Regenerative Finance，再生金融) 机制为核心，汇聚全球资本力量，构建可持续、可扩展的全球绿色经济生态体系，推动绿色产业发展，助力实现联合国 17 项可持续发展目标 (SDGs)。

在此基础上，Green Token 的底层架构将基于全球主流的“绿色金融 ETF”运行逻辑，聚焦优质 ESG 企业与绿色资产，为全球资本提供高效、合规、可规模化参与绿色项目的一站式入口。

通过持续领投与生态化运作，Green Token 将分散的绿色资产整合为可配置、可扩展的绿色资产组合，使生态价值、社会价值与经济价值在同一体系内形成稳定的正向循环。

**核心合作与支持机构👇**

**1. APCED Asia Sdn. Bhd.**

- 成立于 2014 年，马来西亚注册的专业咨询机构
- 聚焦知识产权（IP）评估、无形资产价值管理及 ESG 咨询
- 提供绿色科技、低碳技术及数字化解决方案的资产价值评估
- 协助 XGT 投资项目在 ESG 披露、可持续发展及长期价值管理上形成可核查依据

**2. Allied Environmental Consultants (Malaysia) Sdn. Bhd. (AEC Malaysia)**

- 战略合作机构，隶属香港上市公司 Allied Sustainability and Environmental Consultants Group Limited
- 提供环境与可持续发展专业顾问服务，包括环境影响评估、绿色设计及 ESG 方法论
- 支持项目在能源效率、资源管理、环境绩效及合规披露方面优化与提升

**3. ESG Association of Malaysia（马来西亚 ESG 协会）**

- 马来西亚领先的 ESG 行业协会，推动环境、社会及公司治理发展
- 提供国际 ESG 标准指引、政策解读、培训与教育支持
- 搭建跨界协作平台，促进企业、投资机构及政府部门的交流与合作
